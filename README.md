# 《设计模式的艺术》（C++实现） 

本仓库为《设计模式的艺术》的 C++ 实现，原书的示例代码使用 Java 实现，这里拟使用 C++ 17 重写全书示例，同时完成课后习题。 

## 创建型（Creational）
1. 单例模式
- 示例：负载均衡器（BalanceLoader）
- 练习：数据库连接池（DBConnectionPool） 

2. 简单工厂模式
- 示例：图表库（Chart）
- 练习：几何图形绘图工具（Shape）

3. 工厂方法模式
- 示例：日志记录器（Logger）
- 练习：图片读取器（ImageReader）

4. 抽象工厂模式
- 示例：界面皮肤库（Skin）
- 练习：手机游戏软件（Game）

5. 泛型实现
- 示例：图表库（Chart）
- 练习：无

6. 原型模式
- 示例：OA系统公文管理器（OfficialDocument）
- 练习：无

7. 建造者模式
- 示例：PRG 游戏角色（Actor）
- 练习：视频播放软件（Player）

## 结构型（Structural）
1. 适配器模式
- 示例：教务系统算法库重用（ScoreOperation）
- 练习：OA 系统加密模块（Encryptor）

2. 桥接模式
- 示例：跨平台图像浏览系统（Image）
- 练习：数据转换工具（Converter）
